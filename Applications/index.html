NBA Database Applications

Below you can find the applications for my project. Each individual application has its own php program.


Standings Generator

Pick the whole league, a conference, or a division

Will send you to a page with the results!



Jersey Purchase Menu

Enter a Player's Name and a jersey style (Home, Away, Alternate, or Retro)

And your order will be added to the database


Examples to Test: LeBron James Home, Stephen Curry Retro

Player Information Lookup

Enter a player name, get information about them

Enter a first name in the first field and a last name in the second field


Example names to test: LeBron James, Kevin Durant, Payton Pritchard



College + Arena Lookup

Enter a state, get any college or arena that exists in that state


Example states to test: Oregon, North Carolina, California



Arena Attendance Lookup

Pick a team, get the total attendance at their arena



Team Roster + Salary Information

Pick a team, get the roster, salary of each player, and the average salary

Examples to Test: Portland Trail Blazers, Phoenix Suns
